카테고리별 "다음 액션" 태그를 어떻게 표시할까요?

이메일 수신 시 AI가 카테고리를 분류한 후 — 각 카테고리마다 다음 행동을 Toast에 표시합니다

샘플 이메일 15건 분석 결과 — 카테고리별 주요 Next Action
카테고리 색상 주요 Next Action
Insurance 파란색 답변 필요 / ACK 확인 / 자료 전달
Medical 초록색 서류 제출 / Lien 서명 / 예약 확인
Client 노란색 고객 확인 / 안내 / ACK 없음
Settlement 보라색 변호사 검토 / 오퍼 분석
Police 빨간색 경찰 보고서 수령 / DMV 처리
Other 회색 수동 검토 필요

Next Action 표시 방식을 선택해주세요

A

AI가 자동으로 Next Action 텍스트 생성

Gemini API가 이메일 분류와 함께 "답변 필요", "서류 제출 필요" 같은 한국어 액션 텍스트를 직접 생성. 이메일 내용에 따라 세밀하게 다름.

예: "진술서 요청 — 가능 여부 회신 필요" / "Lien 3건 서명 후 반송 필요"
B

카테고리별 고정 액션 텍스트 (추천)

카테고리에 따라 사전 정의된 액션 표시. Insurance → "답변 필요", Medical → "서류 확인", Client → "고객 확인". 일관성 있고 빠름.

예: Insurance 수신 → 항상 "답변 필요" / Medical 수신 → 항상 "서류 확인"
C

AI 생성 + 카테고리 고정 혼합

카테고리는 고정 액션으로 표시하되, Toast 하단에 AI가 생성한 한 줄 요약도 함께 표시. 정보가 가장 풍부하지만 Toast가 커짐.

예: "답변 필요 — 진술서 가능 여부 + 치료 상황 업데이트 요청"
