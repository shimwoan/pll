동작 흐름 요약

1. Toast 클릭 → matter 상세로 이동, 해당 이메일 핀 카드 자동 표시 + 리스트 자동 접힘
2. 핀 카드 내 Confirm → classification 확정, 핀 카드 사라짐, 리스트 복귀
3. 핀 카드 ✕ 닫기 → 핀 카드 사라짐, 리스트 다시 펼쳐짐
4. 리스트 "펼치기" → 핀 카드 유지하면서 리스트도 함께 표시
5. 리스트에서 다른 이메일 클릭 → 핀 카드가 해당 이메일로 교체
