Toast 클릭 시 이메일 본문 표시 방법

현재: 리스트 → 클릭 → 아래 본문. 문제: 리스트가 공간 차지해서 본문이 안 보임

A

슬라이드오버 드로어 (Slide-over Drawer)

toast 클릭 시 화면 오른쪽에서 드로어가 밀려나오며 이메일 본문을 즉시 표시. 리스트는 뒤에 그대로 유지. 닫기 버튼으로 dismiss.

B

리스트 접기 + 본문 자동 확장 (Collapse List)

toast 클릭 시 이메일 리스트를 자동으로 접고 (헤더만 보이는 collapsed 상태), 해당 이메일 본문 영역을 즉시 스크롤 없이 표시.

C

리스트 위에 고정 본문 패널 (Pinned Preview) ⭐ 추천

toast 클릭 시 리스트 위에 highlight된 본문 카드가 먼저 나타남. 리스트는 그 아래에 축소(compact)된 형태로 유지. 가장 자연스러운 정보 위계.
