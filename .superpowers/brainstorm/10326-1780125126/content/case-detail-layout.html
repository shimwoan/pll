Case Detail 화면 레이아웃

사건 정보 + 체크리스트를 어떻게 배치할까요?

A

접이식 헤더 + 체크리스트

상단에 핵심 정보 카드 (이름, DOL, 보험, 단계)
스크롤하면 접히고 체크리스트가 메인으로 올라옴
체크리스트 집중, 정보는 필요할 때만

B

정보 카드 + 탭 (체크리스트 / 타임라인 / 노트)

상단 고정 정보 카드
탭으로 체크리스트 / 히스토리 / 메모 전환
정보와 작업을 균형있게, 탭으로 전환

C

단계 진행바 + 현재 단계 체크리스트

상단: Claim → CM → Med Collection → Demand 진행바
현재 단계 체크리스트가 메인
핵심 정보는 상단 요약 칩으로
지금 어느 단계인지 + 할 일이 명확
