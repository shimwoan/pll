이메일 업로드 누락

사건전용 이메일주소를 복사해 발송메일에는 cc로, 수신메일은 받는사람에 추가해 전달 → 이메일의 텍스트가 시스템 사건에 자동연동 & 업로드 되고 있음
하지만, 사용법을 모르거나 귀찮아서 안하고 있음
하이퍼링크포함/보안메일은 시스템에서 읽을 수 없음


이메일 연동 방식
● 발송·수신 이메일을 시스템과 연동
→ 사건번호·클레임번호·발신자/수신자 기준으로 사건 자동 매칭

● 매칭 기준 미충족 시 "미분류"로 분류

● 담당자가 자동 분류 결과 확인 후 Accept 또는 Edit 선택 → 해당 사건에 업로드 확정

● 업로드 누락 알림
● 24시간 내 미업로드 시 → 담당자에게 알림
● 24시간 초과 시 → 관리자에게 알림

분류 및 제목 지정
● 업무 유형 Title 드롭다운 제공
: 자주 쓰는 유형 자동 추천
● 직접 입력 옵션 포함

이메일 보기 방식
● 시스템 내 본문 5줄 미리보기 (1차 내용 확인)
● .eml / .msg 파일로 업로드 → 더블클릭 시 Outlook 연동으로 전체 내용 열기
● 보안 메일의 경우 내용 직접 붙여넣기 입력 필드 별도 제공

ACK 관리 (27번 참고)
● ACK 필드: 발송한 이메일 중 ACK 확인이 필요한 경우 체크하고, 이메일별 수신 확인 여부 체크
● 특정 기간 (1-2일) 내 미회신 시 리마인더 발송
